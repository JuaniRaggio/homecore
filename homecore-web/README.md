# HomeCore - Web Prototype

High-fidelity prototype for a smart home management application.

## Requirements

- Node.js 18+
- npm 9+

## Installation

```bash
cd homecore-web
npm install
```

## Running

```bash
npm run dev
```

The dev server starts at `http://localhost:5173`.

## Production Build

```bash
npm run build
npm run preview
```

## Stack

- Vue 3 + Composition API
- Vite
- Vue Router
- Pinia (state management)
- Chart.js (consumption charts)
- Custom CSS (no external frameworks)
